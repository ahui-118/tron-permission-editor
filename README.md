# 🔐 TRON 多签权限管理工具

本工具用于辅助 TRON 用户查看、重置和签署 `accountPermissionUpdate` 多签权限更新交易。

## 功能
- 权限结构 JSON 文件生成
- 签名模板下载
- TronLink 权限更新签名页面
- 支持 2-of-3 多签结构示例

## 使用方法
1. 打开 `index.html` 浏览页面；
2. 下载权限文件并使用签名页面签署；
3. 可部署到 GitHub Pages。
